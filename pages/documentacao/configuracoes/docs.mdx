# Documentação 📃

<Callout type="info" emoji="ℹ️">
  É essencial que o [Node](https://nodejs.org) já esteja instalado em sua
  máquina.
</Callout>

## Passo a passo

Para configurar a documentação na sua máquina, siga os passos abaixo:

<Steps>

### Criar um fork no GitHub

### Clonar o repositório

### Instalar as dependências

Dentro da pasta clonada em sua máquina, execute o comando abaixo:

```shell copy filename="Terminal"
npm install
```

### Executar a aplicação

Execute o ambiente local de desenvolvimento com o comando abaixo:

```shell copy filename="Terminal"
npm run dev
```

</Steps>

<Callout emoji="💡">
  Você pode parar ambiente local de desenvolvimento fechando o terminal ou
  pressionando `CTRL + C` no Windows e Linux, ou `⌘ + .` no macOS.
</Callout>
